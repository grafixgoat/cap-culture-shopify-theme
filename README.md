# CapCLTR Premium Shopify Theme

A professional Shopify Online Store 2.0 theme built for streetwear & lifestyle brands.

## Features

✅ Premium Performance (LCP < 2.5s, CLS < 0.1)
✅ Full Accessibility (WCAG 2.1 AA compliant)
✅ Responsive Mobile Design
✅ Easy Customization (No coding required)
✅ Theme-Native Features (Drops, Badges, Quick Add)
✅ Multi-Currency Support

## Quick Install

### Step 1: Download
Click the green **Code** button above → Select **Download ZIP**

### Step 2: Extract
Unzip the downloaded file to your computer

### Step 3: Upload to Shopify
1. Go to Shopify Admin > **Themes**
2. 2. Click **Upload theme** (top right)
   3. 3. Drag & drop the folder or ZIP file
      4. 4. Wait for upload to complete (~2-3 minutes)
         5. 5. Click **Customize** to edit in theme editor
           
            6. ### Step 4: Publish
            7. Once customized, click **Publish** to make it live
           
            8. ## File Structure
           
            9. ```
               cap-culture-theme/
               ├── config/              # Settings schema
               ├── layout/              # Main page layout
               ├── templates/           # JSON templates (OS 2.0)
               ├── sections/            # Liquid sections
               ├── snippets/            # Reusable components
               └── assets/              # CSS, JavaScript
               ```

               ## Customization

               ### Change Colors
               Theme Editor > Theme Settings > Colors

               ### Add Products
               Shopify Admin > Products > Create product

               ### Create Collections
               Shopify Admin > Collections > Create collection

               ### Configure Metafields
               Admin > Settings > Metafields > Add product badges, descriptions, etc.
